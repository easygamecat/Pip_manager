# PipManager

Графическая утилита (tkinter) для управления установленными pip-пакетами:
просмотр списка, поиск, удаление отдельных пакетов или всех сразу, а также
автоматическая подгрузка краткого описания каждого пакета с PyPI.

## Структура проекта

```
pip_manager/
├── requirements.txt      # зависимости (pyinstaller для сборки)
├── build.bat            # сборка в exe через PyInstaller
├── src/
│   ├── main.py          # точка входа (создаёт Tk root и запускает приложение)
│   ├── core/            # бизнес-логика, без привязки к GUI
│   │   ├── pip_wrapper.py  # работа с pip: список и удаление пакетов
│   │   └── pypi.py        # загрузка описаний с PyPI + кэш
│   └── ui/
│       └── app.py       # PipManagerApp — весь tkinter-интерфейс
```

Принцип разделения: `core/` не знает про GUI, `ui/` пользуется `core/`.
Это упрощает внесение изменений и тестирование логики отдельно от интерфейса.

## Запуск из исходников

```powershell
cd pip_manager
pip install -r requirements.txt
python -m src.main
```

## Сборка в exe

```powershell
build.bat
```
Результат: `dist\PipManager.exe`.

## Основные возможности

- Список всех установленных pip-пакетов с версиями (загружается в фоне)
- Информация о среде: версия Python, venv/системный, путь к интерпретатору
- Поиск по имени
- Группировка похожих имён (например, pyqt4 / pyqt5 объединяются в группу «pyqt»)
- Фильтр «Только группы (конфликты)» — показывает пакеты, у которых есть «соседи» с похожим именем
- Выбор пакетов: обычный клик, Shift/Ctrl, Ctrl+A — выбрать все, Delete — удалить
  (в группировке можно выбрать всю группу — удалятся все пакеты группы)
- «Удалить выбранные» — удаляет отмеченные
- «Удалить все» — удаляет всё с подтверждением
- Колонка «Описание» подгружается с PyPI (с кэшированием)
